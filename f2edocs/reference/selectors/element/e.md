#类型选择符(E)

##语法

- E { sRules }


##说明

以文档语言对象类型作为选择符。

- 类型选择符(Type Selector)

##示例

```html

<!DOCTYPE html>
<html lang="zh-cmn-Hans">
<head>
<style>
h1 {
	font-size: 20px;
}
p {
	font-size: 13px;
}
</style>
</head>
<body>
<h1>标题</h1>
<p>正文内容</p>
</body>
</html>

```
