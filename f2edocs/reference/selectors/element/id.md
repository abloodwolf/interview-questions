#ID选择符(E#id)

##语法

- E#myid { sRules }


##说明

以唯一标识符id属性等于myid的E对象作为选择符。

- ID选择符(ID Selector)

##示例

```html

<!DOCTYPE html>
<html lang="zh-cmn-Hans">
<head>
<style>
#subtitle {
	font-size: 20px;
}
p#content {
	font-size: 13px;
}
</style>
</head>
<body>
<h1 id="subtitle">标题</h1>
<p id="content">正文内容</p>
</body>
</html>

```
