#@page :right

##语法

- @page :right { sRules }


##说明

设置页面容器位于装订线右边的所有页面使用的样式。仅用于@page规则。

- 该伪类选择符只允许定义margin, padding, border 和 background属性
- 相关查看：@page, :first, :left
