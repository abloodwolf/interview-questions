#E:not(s)

##语法

- E:not(s) { sRules }


##说明

匹配不含有s选择符的元素E。

- 有了这个选择符，那么你将可以很好的处理类似这样的场景：假定有个列表，每个列表项都有一条底边线，但是最后一项不需要底边线


**示例：**

```css
.demo li:not(:last-child) {
	border-bottom: 1px solid #ddd;
}
```

>上述代码的意思是：给该列表中除最后一项外的所有列表项加一条底边线
