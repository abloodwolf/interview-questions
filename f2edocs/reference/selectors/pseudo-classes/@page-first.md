#@page :first

##语法

- @page :first { sRules }


##说明

设置在打印时页面容器第一页使用的样式。仅用于@page规则。

- 该伪类选择符只允许定义margin, orphans, widows 和 page breaks相关属性
- 相关查看：@page, :left, :right

##示例

```html

<!DOCTYPE html>
<html lang="zh-cmn-Hans">
<head>



<style>
@page :first {
	margin: 100px 200px;
}
div {
	page-break-after: always;
}
</style>
</head>
<body>
<div>页面容器中第一页的外边距为100px 200px</div>
<div>页面容器中第二页的外边距为默认值</div>
</body>
</html>

```
