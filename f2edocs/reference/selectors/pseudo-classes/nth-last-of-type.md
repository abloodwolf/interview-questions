#E:nth-last-of-type(n)

##语法

- E:nth-last-of-type(n) { sRules }


##说明

匹配同类型中的倒数第n个同级兄弟元素E。

- 要使该属性生效，E元素必须是某个元素的子元素，E的父元素最高是html，即E可以是html的子元素，也就是说E可以是body
- 该选择符总是能命中父元素的倒数第n个为E的子元素，不论倒数第n个子元素是否为E
