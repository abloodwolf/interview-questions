#E:checked

##语法

- E:checked { sRules }


##说明

匹配用户界面上处于选中状态的元素E。(用于input type为radio与checkbox时)

##示例

```html

<!DOCTYPE html>
<html lang="zh-cmn-Hans">
<head>
<style>

	background: #f00;
}

	content: " 我被选中了";
}
</style>
</head>
<body>
<form method="post" action="#">
<fieldset>
	<legend>选中下面的项试试</legend>
```
