#E:lang(fr)

##语法

- E:lang(fr) { sRules }


##说明

匹配使用特殊语言的E元素。

##示例

```html

<!DOCTYPE html>
<html>
<head>
<style>
p:lang(zh-cmn-Hans) {
	color: #f00;
}
p:lang(en) {
	color: #090;
}
</style>
</head>
<body>
<p lang="zh-cmn-Hans">大段测试文字</p>
<p lang="en">english</p>
</body>
</html>

```
