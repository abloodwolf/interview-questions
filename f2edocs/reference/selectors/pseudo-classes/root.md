#E:root

##语法

- E:root { sRules }


##说明

匹配E元素在文档的根元素。在HTML中，根元素永远是HTML

- 根据这个特性，可以做IE8的Hack


**示例：**

```css
.test {
	color: black;
	color: yellow\0;
	*color: blue;
	_color: red;
}
html:root .test {
	color: purple\0;
}
```

>上述代码：非IE文本将为black，IE9及以上为purple，IE8为yellow，IE7为blue，IE6为red
