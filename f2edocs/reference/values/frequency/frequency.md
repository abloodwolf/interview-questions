#&lt;frequency&gt;

##说明

&lt;number&gt;接频率单位。

- 频率值不允许有负值。
- 频率单位包括有：Hz, kHz
- 举例说明：200Hz(或200hz)表示一个低音，6kHz(或6khz)表示一个高音。
- 单位换算：1kHz = 1000 Hz
