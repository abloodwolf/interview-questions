#unset

##说明

擦除属性申明。

- 一个属性定义了unset值，如果该属性是默认继承属性，该值等同于inherit，如果该属性是非继承属性，该值等同于initial
- unset属于CSS-wide关键字，这表示所有的属性都可以接受该值，本手册为了与规范保持一致，未将该值放置到各属性的取值中。
- 正常工作的例子：


**示例代码：**

```css
div {
	border: unset;
}
div {
	border: initial;
}
```

>如上述代码，2条规则的结果是相同的
