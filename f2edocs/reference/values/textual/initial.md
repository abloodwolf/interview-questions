#initial

##说明

属性初始值。

- initial属于CSS-wide关键字，这表示所有的属性都可以接受该值，本手册为了与规范保持一致，未将该值放置到各属性的取值中。
- 如果你想重置某个属性为UA默认设置，那么就可以使用该值
- 正常工作的例子：


**示例代码：**

```css
ul {
	list-style: none;
}
.news-list {
	list-style: initial;
}
```

>如上述代码，ul会被reset掉list-style，如果你想某个地方需要UA初始定义，那么就可以直接设置为initial
