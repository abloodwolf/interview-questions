#&lt;string&gt;

##说明

字符串(含转义字符串)。

- 字符串可以由双引号或单引号包起来。
- 转义字符串由 反斜杠(\) 开始，如：\', \"
- 2个常规字符串例子。请看下述代码：


**示例代码：**

```css
div:after {
	content: "任意字符串";
}
a[title="external"] {
	/*...*/
}
```

- 2个转义字符串例子。请看下述代码：


**示例代码：**

```css
div:after {
	content: "任意字符串\"String\"";
}
div {
	/* \5b8b\4f53 = 宋体 */
	font-family:'\5b8b\4f53';
}
```
