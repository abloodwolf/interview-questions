#inherit

##说明

该值使得属性能够继承祖先设置。

- inherit属于CSS-wide关键字，这表示所有的属性都可以接受该值，本手册为了与规范保持一致，未将该值放置到各属性的取值中。
- 如何让一个不具备继承特性的属性可以继承父元素的定义？


**示例代码：**

```css
div {
	position: relative;
}
div a {
	position: inherit;
}
```

>上述代码，超链接 a 将会继承父元素的position定义，也会定义为relative
