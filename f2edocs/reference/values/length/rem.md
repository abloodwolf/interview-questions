#rem

##说明

相对长度单位。相对于根元素(即html元素)font-size计算值的倍数

##示例

```html

<!DOCTYPE html>
<html lang="zh-cmn-Hans">
<head>
<style>
html,
h1 {
	font-size: 12px;
}
p {
	font-size: 2rem;
}
</style>
</head>
<body>
<h1>下面的文字将是html定义的字体大小的2倍：</h1>
<p>我是html定义的12px的2倍，字体大小为24px</p>
</body>
</html>

```
