#vmax

##说明

相对于视口的宽度或高度中较大的那个。其中最大的那个被均分为100单位的vmax

##示例

```html

<html lang="zh-cmn-Hans">
<head>
<style>
p {
	font-size: 5vmax;
}
</style>
</head>
<body>
<p>相对于viewport的宽度或高度中较大的那个然后计算文字大小。</p>
</body>
</html>

```
