#&lt;percentage&gt;

##说明

&lt;number&gt;接%

- 百分比总是相对于另外一个值才存在的。


示例代码：

```css
div {
	font-size: 12px;
}
.test {
	font-size: 200%;
}
```


- 百分比的单位是：%
