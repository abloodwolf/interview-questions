#column-width

##语法

- column-width：&lt;length&gt; | auto
- 默认值：auto
- 适用于：除table外的非替换块级元素, table cells, inline-block元素
- 继承性：无
- 动画性：否
- 计算值：绝对长度值，0或者更大


##取值

- &lt;length&gt;：用长度值来定义列宽。不允许负值
- auto：根据 &lt;' column-count '&gt; 自定分配宽度


##说明

设置或检索对象每列的宽度

- 对应的脚本特性为columnWidth。
