#columns

##语法

- columns：&lt;' column-width '&gt; || &lt;' column-count '&gt;
- 默认值：看每个独立属性
- 适用于：除table外的非替换块级元素, table cells, inline-block元素
- 继承性：无
- 动画性：否
- 计算值：看每个独立属性


##取值

- &lt;' column-width '&gt;：设置或检索对象每列的宽度
- &lt;' column-count '&gt;：设置或检索对象的列数


##说明

设置或检索对象的列数和每列的宽度。复合属性

- 对应的脚本特性为columns。
