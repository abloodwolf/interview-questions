#column-rule-color

##语法

- column-rule-color：&lt;color&gt;
- 默认值：采用文本颜色
- 适用于：定义了多列的元素
- 继承性：无
- 动画性：否
- 计算值：与 &lt;' color '&gt; 属性一致


##取值

- &lt;color&gt;：指定颜色。


##说明

设置或检索对象的列与列之间的边框颜色。

- 如果 &lt;' column-rule-width '&gt; 等于0或 &lt;' column-rule-style '&gt; 设置为none | hidden，本属性将被忽略。
- 对应的脚本特性为columnRuleColor。
