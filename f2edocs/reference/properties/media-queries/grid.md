#Media Queries: grid

##语法

- grid：&lt;integer&gt;


##取值

- &lt;integer&gt;：用整数值来定义是否使用栅格或点阵。只有1和0才是有效值，1代表是，0代表否


##说明

用来查询输出设备是否使用栅格或点阵。

- 本特性不接受min和max前缀。

**简单列举几个应用示例：**

```css
@media all and (grid){ … }
@import url(example.css) all and (grid:0);
```
