#border-top-colors

##语法

- border-top-colors：&lt;color&gt;{1,}
- 相关属性：&lt;' border-right-colors '&gt; || &lt;' border-bottom-colors '&gt; || &lt;' border-left-colors '&gt;
- 适用于：所有元素
- 继承性：无
- 动画性：是
- 计算值：指定值


##取值

- &lt;color&gt;：指定颜色。


##说明

设置或检索对象顶部边框的多重颜色。

- border的width为(n)px，那么最多可以设置n组边框色，每组边框色宽度为1px；如果border的width为10px，却只设置了6组边框色，那么最后一组边框色将自动渲染剩余的宽度
- 对应的脚本特性为borderTopColors。

<html lang="zh-cmn-Hans">

</head>
